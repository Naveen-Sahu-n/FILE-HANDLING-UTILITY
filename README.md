## This project is a console-based Java application developed as part of the CodTech Solutions internship. It showcases the implementation of file handling techniques using Java’s java.io package.
The system allows users to interact with text files (.txt) through a structured menu. Users can perform multiple operations like creating a file, writing data, reading content, appending new data, and modifying existing content by replacing specific words.

Language: Java
Packages Used:
java.io.File
FileWriter
FileReader
BufferedReader
java.util.Scanner

Working 
The user is prompted to enter a valid .txt file name.
A menu is displayed with multiple file operation options.
Based on user input, the corresponding operation is executed.
Exception handling ensures smooth execution without crashes.
